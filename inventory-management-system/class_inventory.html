Detailed Description

Manages a collection of products in the inventory.

The Inventory class provides methods to add products, save the inventory to a CSV file, load products from a CSV file, and display the list of products in the inventory.

◆ loadFromCsv()

void Inventory::loadFromCsv ( const string & filename)

Loads product data from a CSV file.

This method reads product information from a CSV file and populates the internal list of products. It assumes the CSV file follows the format: id,productType,model, seriesNumber,brandName,state. The first line of the file is skipped as it contains the header.

Parameters
filenameThe name of the CSV file to load the data from.
Note
The existing products in the inventory are cleared before loading new data.

Saves the inventory to a CSV file.

Parameters

Note
